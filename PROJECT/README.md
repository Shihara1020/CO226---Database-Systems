# CO226 Database Project

This folder contains the main deliverables for the CO226 database project.

## Contents

- `CO226_E21_Apr2025_SQL_G26.sql` - Main SQL script for schema and/or data operations.
- `CO226_E21_Apr2025_ER_G26.pdf` - Entity Relationship (ER) diagram.
- `CO226_E21_Apr2025_ERDescription_G26.pdf` - ER model description document.
- `CO226_E21_Apr2025_LogicalDiagram_G26.pdf` - Logical design diagram.
- `Er.drawio` - Editable ER diagram source file.
- `Untitled Diagram.drawio` - Additional editable diagram source.

## How to Use

1. Open your SQL client (for example, MySQL Workbench or phpMyAdmin).
2. Create/select the target database.
3. Run `CO226_E21_Apr2025_SQL_G26.sql`.
4. Review the PDF and draw.io files for design references.

## Notes

- Keep diagram sources (`.drawio`) updated when modifying the schema.
- Keep the SQL file and diagrams synchronized.
