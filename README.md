## What I Learned (Labs)

- Writing SQL queries: filtering, sorting, aggregation (`GROUP BY` / `HAVING`).
- Joins and subqueries for combining data across tables.
- Schema design with DDL: `CREATE TABLE`, primary/foreign keys, and constraints.
- Mapping ER models to relational schemas (and keeping diagrams consistent with SQL).
- Loading datasets using SQL scripts (example: employee dataset in Lab 08).
- Basic database-backed web development using Django + SQLite.

## Quick Start

1. Open this folder in VS Code.
2. Pick the relevant lab folder (LAB02–LAB08) or the [PROJECT](PROJECT/) folder.
3. Run the SQL scripts in your DBMS (MySQL / PostgreSQL / SQLite) using your preferred client.
4. Open `.drawio` files to edit diagrams and export to PDF if needed.
